Full Stack Development

Hello! I'm Sajwal Chaulagain, a 1st-year student studying Full Stack Development. This ePortfolio showcases my learning journey in CET138, where I have explored HTML, CSS, Bootstrap, and JavaScript.

Full stack development means building both the front-end, which the user sees, and the back-end, which handles the server, database, requests, data processing, and application logic.

I understand that a full stack developer connects design, functionality, and data together to create complete modern web applications.

Front-End

The browser part built with HTML, CSS, and JavaScript. It shows content and handles user interactions.

Back-End

The server side that processes requests, checks logins, runs logic, and communicates with databases.

Database

Stores data such as users, products, orders, messages, and other application information.

Request-Response Cycle

The browser sends a request to the server. The server processes it, may ask the database for data, and sends a response back.

HTML

HTML means HyperText Markup Language. It is the building block of every webpage and is used to structure content such as headings, paragraphs, images, links, lists, tables, forms, and sections.

I understand that modern HTML5 includes semantic tags like <header>, <nav>, <main>, <article>, and <footer>. These tags make a webpage easier for browsers, search engines, and screen readers to understand.

<header>
  <h1>Sajwal Chaulagain</h1>
  <nav>Home | About | Contact</nav>
</header>

<main>
  <article>
    <h2>My First Article</h2>
    <p>This is structured HTML content.</p>
  </article>
</main>

CSS

CSS means Cascading Style Sheets. It is used to style HTML elements and control colours, fonts, spacing, layout, borders, hover effects, animations, and responsive design.

I understand that CSS works by selecting an HTML element and applying rules to it. CSS can also use reusable variables and the box model to control page layout properly.

h1 {
  color: purple;
  font-size: 36px;
  text-align: center;
}

.card {
  background-color: white;
  border: 1px solid #ddd;
  padding: 20px;
  border-radius: 8px;
}

:root {
  --main-color: #6c63ff;
}

Bootstrap Framework

Bootstrap is a free front-end CSS framework that provides ready-made styles and components. It helps developers build responsive and professional websites quickly without writing every style from scratch.

I understand that Bootstrap uses classes for layout and design. Its 12-column grid system makes it easier to create pages that work on desktop, tablet, and mobile screens.
